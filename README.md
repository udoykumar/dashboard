# Dashboard

A modern React + Tailwind CSS dashboard with login

## Tech Stack

- **React 19** (Vite)
- **Tailwind CSS v4**
- **React-Router**
- **Axios**
- **API**: `https://task-api-eight-flax.vercel.app`

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Features

- ✅ JWT Authentication (localStorage persistent)
- ✅ Login / Logout
- ✅ Fetch & display
- ✅ Toast notifications
- ✅ Fully responsive (mobile sidebar)
- ✅ Smooth animations

## Deploy (Vercel)

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Deploy — done!
